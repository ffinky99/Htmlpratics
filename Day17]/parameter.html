자바 스크립트의 파라미터의 특징과 기능 자바에서 매개변수를 자바 스크립트에서 파라미터 라고 함.

  • 파라미터의 개수와 관계없이 함수나 매서드의 이름이 같다면 호출이 가능.
  • 파라미터의 개수가 불일치 하여 값을 받지 못하는 파라미터는 undifined 처리
  • 값을 받아오지 못하는 파라미터가 undifiend 일 경우 별도로 파라미터에 기본값을 처리 할 수 있음.=> optional parameter
  • optional parameter : 파라미터의 값이 전달되지 않았을 때 기본값을 설정해주는 파라미터
  • optional parameter 는 파라미터 중 가장 끝에 위치해야 함
  • 모든 파라미터를 관리하는 arguments 라는 파라미터를 이용하여 파라미터를 배열에 담아 사용할 수 있음
  • 단 optional parameter는 저장하지 않는다.
  • rest parameter : 나머지(이외의, 남는것들)파라미터
  • 사용할 파라미터는 일반 파라미터로 배정하고, 나머지 파라미터들을 통합하여 하나의 이름으로 배정하는 기법
