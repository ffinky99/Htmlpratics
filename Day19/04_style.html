Css 적용은 element.style.property형식으로 접근이 가능

  • property는 자바스크립트의 코드 : -(하이픈) 적용 안됨
  • js에서는 css 속성의 - 하이픈을 제거하고, 카멜표기법으로 변경
  • Element.setAtrrubute(name,value) : element에 속성뷰ㅜ여 이미 있는 속성이라면 업데이트, 없으면 새 속성으로 추가
  • cssName.setProperty(name,value) : css 스타일을 부여할 대상
  • name : css Name, value : 속성값
