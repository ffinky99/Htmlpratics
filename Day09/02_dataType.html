변수나 상수에 할당할 수 있는 값의 종류

  • 숫자 : Number => 정수, 실수, NaN(not a number : 숫자가 아님)
  • 문자 : String => " ", ' ',` ` 감싼 값들
  • boolean : True / false
  • 객체 : Object => 1개 이상의 데이터가 특정 구조를 가지고 있는 형태
  • 배열 : Array : [1,2,3,...] 형태로 이루어진 데이터 집합
  • 알 수 없는 값 : null => 뭔가 있긴한데 파악이 안됨
  • 미지정된 값 : undefined => 값이 할당되지 않음
  • 데이터 타입을 확인 명령어 : typeof 변수 / 값

    • 변수의 관리

    1. 기본자료형(Primative Type): 값이 메모리의 주소값이 가르키는 곳에 직접 할당
    2. 참조 자료형 (Refrence Type) : 값이 저장되어 있는 주소를 할당 String, object, array, function ,null nudefiend
    3. ==같다(값) != 같지않다 ===같다 (값+타입) !==같다 (값과 타잆이 같지 않은지 비교)
