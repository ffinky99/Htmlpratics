• html 태그의 속성이 제한적임, 사용자 들에게 표준이 아닌 사용자만의 문법을 설계할 수 있도록 하는 속성
  • html 속성이 제한적이므로 데이터 관리 차원에서 개발자들에게 직접 속성을 만들어서 사용할 수 있는 방법을 제공
  • 너무 과도하게 사용하면 개발 효율과 가독성이 낮아질 수 있으므로 적당히 사용하는 것을 권장
  • 태그의 속성 이름을 data- 로 싲가하고 하이픈(-) 뒤에 특정 의미의 이름을 사용 할 수 있음
  • 이 규칙을 자바스크립트에서 인식 할 수 이쏘록 설계되어 있으
  • dataset.name 앍오둘암 name에 대문자 허용안됨 만약 대문자 넣으면 소문자로 변환해서 사용하면 ㄱㄴ
